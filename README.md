CyberToolkit

CyberToolkit is a Python-based cybersecurity toolkit project developed as a modular desktop application. It focuses on GUI development, basic security tool concepts, and team-based software engineering practices.

---

Features

- Secure login system (basic authentication)
- Graphical User Interface (Tkinter-based)
- Modular cybersecurity tools structure
- Team-based development workflow
- Basic logging and data handling system

---

Setup Instructions

Requirements

Make sure you have the following installed:

- Python 3.10 or above
- Git
- VS Code (recommended)

---

Install Required Libraries

pip install requests

---

Installation Steps

1. Clone the repository

git clone https://github.com/AmnaSaaim/CyberShieldToolkit.git
cd cybertoolkit

---

2. Create virtual environment (recommended)

python -m venv venv

---

3. Activate virtual environment

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

---

4. Run the project

python main.py
